# JavaScript CRUD Application (with LocalStorage)

A lightweight, vanilla JavaScript application demonstrating full **CRUD** (Create, Read, Update, Delete) operations using the browser's `localStorage` for persistent data management.

## 🚀 Features

*   **Create:** Add new records easily via a user-friendly form.
*   **Read:** Display stored data dynamically in a clean table or list layout.
*   **Update:** Edit existing entries and save changes instantly.
*   **Delete:** Remove records individually (with optional confirmation).
*   **Persistent Storage:** Data survives page refreshes thanks to `localStorage`.

## 🛠️ Tech Stack

*   **HTML5:** Structured semantic markup.
*   **CSS3:** Clean layout and responsive UI styling.
*   **JavaScript (ES6+):** Core logic, DOM manipulation, and `localStorage` API handling.
